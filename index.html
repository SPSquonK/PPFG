Criteria for a playable fan game

My main criteria to play a game are the following:

  • Editable: I won't play if I can't apply some QOL changes (visible Hidden Items, every Pokemon have the pickup ability, remap keys, Gen 6 Share Exp...)
  • Gen 3 battle layout: Other battle layouts are too slow
  • Speedup feature: Pokemon games are slow so the speed up feature is vital. Note that if the game is editable, I can add myself the speedup feature.
  • Finished and/or with enough accessible content: I won't play 1 hour long demo
  • A regular Pokemon game: So not Mystery Dungeon, not a game that uses the Pokemon universe to propose a different type of game

It's also better if you let me play quickly. Exposure before the first fight of the game is useless because until I can pick a Pokemon and see a battle, I'm not sure that I will keep playing.
